# Nvchad_custom

# Usage

## Install Nvchad

```sh
git clone https://github.com/NvChad/NvChad ~/.config/nvim --depth 1
```

## Clone Repo

```sh
git clone git@github.com:LinFight/nvchad_custom.git ~/.config/nvim/lua/custom
```

## Backup Old Config

```sh
mv ~/.config/nvim/lua/custom ~/.config/nvim/lua/custom.bak
```
